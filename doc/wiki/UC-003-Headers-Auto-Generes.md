# UC-003: Headers Auto-générés

## Vue d'ensemble

Visualisez les headers ajoutés automatiquement par l'application selon votre configuration.

## Comment visualiser

1. Dans l'onglet **Headers**, trouvez **"Auto-generated Headers"**
2. Cliquez sur **"Show"** pour déplier
3. Les headers s'affichent avec leur **source**

## Types de headers auto

### Système (toujours présents)

| Header          | Valeur              |
| --------------- | ------------------- |
| User-Agent      | Rune-API-Client/1.0 |
| Accept          | _/_                 |
| Accept-Encoding | gzip, deflate, br   |
| Accept-Language | en-US,en;q=0.9      |
| Connection      | keep-alive          |

### Liés au Body

| Condition       | Header         | Valeur               |
| --------------- | -------------- | -------------------- |
| Body = raw      | Content-Type   | Selon rawContentType |
| Body = formData | Content-Type   | multipart/form-data  |
| Body présent    | Content-Length | Taille en bytes      |

### Liés à l'Auth

| Type d'auth | Header        | Valeur         |
| ----------- | ------------- | -------------- |
| Bearer      | Authorization | Bearer {token} |
| Basic       | Authorization | Basic {base64} |

## Indication de source

- **from System** : Ajouté par défaut
- **from Body** : Déduit du type de body
- **from Auth** : Injecté depuis l'onglet Auth

## Désactivation

Vous pouvez désactiver un header auto spécifique :

1. Trouvez le header dans la liste
2. **Décochez** la case
3. Le header ne sera pas envoyé

## Différence manuel vs auto

| Caractéristique | Auto              | Manuel      |
| --------------- | ----------------- | ----------- |
| Création        | Automatique       | Utilisateur |
| Modification    | Via configuration | Directe     |
| Apparence       | Grisé/opacité     | Normal      |
